# Features Overview

Agent Desk packs a comprehensive set of features designed for developers who work with AI coding agents. Here's a high-level comparison with generic terminal emulators.

## Feature Comparison

| Feature | Agent Desk | Generic Terminals |
|---------|-----------|-------------------|
| Tabbed terminals | Yes | Yes |
| Split grid layout | Yes | Some |
| Drag-drop tabs | Yes | Some |
| Pop-out windows | Yes | Rare |
| Agent detection | Yes | No |
| Agent status tracking | Yes | No |
| Tool call monitoring | Yes | No |
| Cost/token tracking | Yes | No |
| Batch agent launch | Yes | No |
| Agent templates | Yes | No |
| Cross-terminal search | Yes (async, regex) | Rare |
| Event timeline | Yes | No |
| Dashboard integration | Yes (3 services) | No |
| Communication graph | Yes | No |
| Workspace save/load | Yes | Some |
| Custom themes (8+) | Yes | Varies |
| Customizable keybindings | Yes | Yes |
| Shell integration (OSC) | Yes | Some |
| System resource monitor | Yes (CPU/RAM/disk) | No |
| Command palette | Yes | Some |
| Quick switcher | Yes | Some |

## Feature Categories

### [Terminal Management](/features/terminals)
Multi-terminal grid with tabs, splits, drag-drop, pop-out, and powerful search.

### [Agent Intelligence](/features/agents)
Automatic agent detection with status tracking, tool call monitoring, file modification logging, and cost metrics.

### [Multi-Agent Orchestration](/features/orchestration)
Batch launcher, templates, terminal chains, and the agent monitor view for coordinating parallel agent workflows.

### [Search & Events](/features/search-events)
Cross-terminal search with regex support, plus a real-time event stream with filtering and export.

### [Appearance](/features/appearance)
8 built-in themes (4 dark + 4 light), custom theme editor, system theme following, and dashboard theme sync.

### [Session & Persistence](/features/sessions)
Workspace save/load, session persistence, config file sync, and auto-update support.
